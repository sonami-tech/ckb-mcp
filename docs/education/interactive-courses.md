# CKB Interactive Learning Courses

## Description

Curated collection of interactive CKB blockchain development courses, tutorials, and hands-on learning resources. Features official Nervos training programs, community workshops, coding challenges, and practical exercises for different skill levels. Includes step-by-step tutorials, video content, interactive coding environments, and certification programs. Essential learning path for developers new to CKB development and those advancing their blockchain programming skills.

## CKB Academy Interactive Courses

The CKB Academy provides structured, hands-on learning experiences for developers at all levels. These courses combine theoretical knowledge with practical implementation.

### Available Interactive Courses

#### 1. CKB Basic Theoretical Knowledge
**Course ID**: `basic-theory`
**Target Audience**: Beginners to CKB development
**Duration**: 9 chapters
**Format**: Interactive web-based learning

**Course Content**:
- **Chapter 1**: What is CKB? - Understanding the cell-based architecture
- **Chapter 2**: Cell Data Structure - Capacity, lock, type, and data fields  
- **Chapter 3**: Cell Ownership - How lock scripts determine ownership
- **Chapter 4**: Summary 1 - Consolidating basic concepts
- **Chapter 5**: Code Location - Where smart contract code is stored
- **Chapter 6**: Code Preservation - Handling lost lock code scenarios
- **Chapter 7**: Transaction Fundamentals - How state changes work
- **Chapter 8**: Type Script Role - Understanding type script functionality
- **Chapter 9**: Summary 2 - Complete conceptual overview

**Key Learning Outcomes**:
- Understand CKB's cell model vs traditional account model
- Learn cell data structure and capacity management
- Grasp the relationship between cells and transactions
- Understand lock and type script roles

#### 2. CKB Basic Practical Operation
**Course ID**: `basic-operation`  
**Target Audience**: Developers ready for hands-on practice
**Duration**: 3 chapters
**Format**: Interactive with wallet connection and testnet transactions

**Course Content**:
- **Chapter 1**: Connecting Wallet - Integrate with CKB testnet wallets
- **Chapter 2**: Transaction Theory - Deep dive into transaction mechanics
- **Chapter 3**: Send Transaction - Manually construct and send testnet transactions

**Key Learning Outcomes**:
- Connect to CKB testnet using supported wallets
- Understand transaction construction and validation
- Successfully send transactions to CKB testnet
- Verify transactions using CKB explorer

#### 3. Getting Started With NFTs
**Course ID**: `nft-getting-started`
**Target Audience**: Developers interested in NFT development
**Duration**: 6 chapters  
**Format**: Article-based with comprehensive comparisons

**Course Content**:
- **Chapter 1**: Introduction to CKB NFT ecosystem
- **Chapter 2**: Standards Overview - CoTA vs Spore comparison
- **Chapter 3**: CoTA Strengths - Cost-effective batch NFTs
- **Chapter 4**: Spore Strengths - Fully on-chain permanent NFTs
- **Chapter 5**: Standard Comparison - Choosing the right approach
- **Chapter 6**: Next Steps - Implementation guidance

**Key Learning Outcomes**:
- Understand different NFT standards on CKB
- Choose appropriate NFT standard for specific use cases
- Learn implementation patterns for both CoTA and Spore
